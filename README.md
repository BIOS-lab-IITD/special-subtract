# special-subtract
Special subtract images in a system folder and output difference images.  
This is helpful in a simple image processing task to supress static information, and highlight dynamic information. It works with a time series image sequence.
We use it extensively for Consecutive Frame Subtraction (CFS) in schlieren imaging. 


## How to use?
1. Download the latest release (.exe file) for windows. 
2. Double click it
3. Enter the folder path containing the time series images, and desirable output folder paths.

4. For other OS, build using the source provided here!
